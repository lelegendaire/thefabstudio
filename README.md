## ✨ About

The Fab Studio is a french creative web studio born from a simple yet powerful idea: building websites that are truly unique, tailored to every person. Founded in 2023 by **Fabien Chevallereau**, the studio sits at the intersection of code, aesthetics, and interactive storytelling.

> *"A visual laboratory where innovation, aesthetics and interactivity meet to invent unique experiences."*

---

## 🗂️ Project Structure

```
thefabstudio/
├── app/          # Next.js App Router (pages, layouts, routes)
├── components/   # Reusable UI components
├── context/      # React context providers
├── lib/          # Utility libraries and helpers
├── public/       # Static assets (images, fonts…)
└── utils/        # Utility functions
```





## 🛠️ Tech Stack

- **Framework** — [Next.js](https://nextjs.org) (App Router)
- **Styling** — CSS / Tailwind CSS
- **Deployment** — [Vercel](https://vercel.com)
- **Language** — JavaScript
